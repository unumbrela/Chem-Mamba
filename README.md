# Chem-Mamba

**研究机器学习势能模型（MLIP）是否真正学到非局域电荷转移物理：
判别性评测协议 + 受控模型矩阵 + 训练方法。**
（仓库名沿用项目早期定位，SSM/Mamba 现在是本项目七个受控变体之一。）

## 课题（三句话）

1. 真实体系里存在非局域电荷转移：在 10 埃以外掺入一个杂质原子，可以改变这里的
   金团簇带不带电（Au₂@MgO 基准，Ko et al., *Nat. Commun.* 2021）。主流 MLIP
   假设每个原子只受几埃内邻居影响，对这类现象结构上无法表达。
2. 现行评测判断不了模型是否学到了这类物理：能量误差、力误差、电荷误差、
   电荷相关系数都可以在电荷物理完全错误的情况下同时变好（本仓库的受控实验证明）。
3. 本项目提供能判断它的工具：匹配判别对（局域环境相同、只有远场不同的成对构型）、
   空间对比度指标（对"把电荷均匀分摊"类捷径的贡献严格为零）、容量匹配对照
   （参数相同、只切断非局域信息流的对照模型），以及让电荷物理训练得出来的
   两阶段训练方法。

## 受控模型矩阵

所有变体共享同一骨干网络、数据划分与训练协议，只在"非局域信息怎么进入电荷"上不同：

| 变体 | 非局域形态 | 复杂度 |
|---|---|---|
| `local-Q` | 纯局域（= LES/3G 类方法的结构等价物） | O(N) |
| `local+Q` | 局域 + 总电荷标量 | O(N) |
| `ssm-iso` | 无跨原子信息流（容量匹配对照，参数与 ssm 相同） | O(N) |
| `ssm` | 双向选择性 SSM 扫描 | O(N) |
| `ssm2` | 每层 summary-token（聚合再分配结构） | O(N) |
| `attn` | 全注意力、无截断（能力上界参照） | O(N²) |
| `qeq` | 可微电荷均衡（固定物理假设参照） | 解线性系统 |

骨干可切换：SchNet 风格不变特征（默认）或 MACE 等变骨干（`--backbone mace`）。
长程静电用可微 Ewald 求和（含板层体系的 Yeh–Berkowitz 偶极修正），
验证到 Madelung 常数误差 2.6e-6、解析力与数值差分一致到 3e-11。

## 目前的主要结果（单种子，数据集 = Ko et al. 2021 四基准）

- **表达能力**：电荷作辅助输出时，O(N) 的 SSM 通道在 AuMgO 上把电荷从 10 埃外的
  掺杂位置分配到 Au 团簇，空间对比度 −132 me = DFT 真值（me = 千分之一电子电荷）；
  参数完全相同、只切断全局信息的对照只有 −30。纯局域模型严格为零。
- **指标失效**：NaCl 单变量消融——加大截断、加大力权重、拉长调度、EMA 权重平均，
  单独任何一个都把判别性电荷指标从正确变成符号错误或接近零，同时能量和力
  持平或变好。电荷相关系数同样会给出错误结论（实测 corr 0.613 与对比度 ≈0 并存）。
- **训练脆弱性与解法**：电荷接入静电能量后，从头训练学不到空间电荷分配
  （30k 步对比度仍 ≈0，此时能量/力/电荷误差都已明显下降）；
  先用电荷标签训练、再接入能量继续训练（同一批标签，只改顺序），
  精度追平从头训练且对比度恢复到 −131（真值 −132）。
  同框架的全注意力模型无此优化障碍——该现象与架构有关。
- **诚实对照**：Ag₃⁺/Ag₃⁻ 上（总电荷就是全部非局域信息）所有全局变体打平，
  SSM 无虚假增益；碳链上电荷精度达到 4G-HDNNP 文献区间，成本 O(N) 而非 O(N³)。

## 概念验证（`poc/`，1D）

在可控的电荷均衡（QEq）任务上隔离检验一件事：局域模型在"局域环境冻结、
只变远场"的判别测试上是否结构性失效，全局模型是否能通过。

| 模型 | 复杂度 | 判别测试相关性 |
|---|---|---|
| local（局域 DeepSet） | O(N) | −0.00（输出为常数，结构性失效） |
| ssm（双向 Mamba S6） | O(N) | 0.896 |
| transformer（全注意力） | O(N²) | 0.466 |

标度实测：SSM 时间/显存随 N 线性；全注意力显存二次，16384 原子时 8.6 GB。

```bash
cd poc
PYTHONPATH=. python run_poc.py      # 主实验（约 10 分钟）
PYTHONPATH=. python scaling.py      # 标度实验（约 2 分钟）
```

## 3D 代码（`chem_mamba/`）

`layers.py`（SSM 块）· `model4g.py`（七变体模型矩阵）· `ewald.py`（可微 Ewald）·
`mace_backbone.py`（MACE 骨干适配）· `runner_data.py`（Ko 基准解析）·
`train_4g.py`（训练，含两阶段训练、对称性检查、判别指标评估）·
`summarize.py`（结果汇总）。

```bash
PYTHONPATH=. python chem_mamba/train_4g.py --dataset Carbon_chain --model ssm
```

对称性验证：能量对旋转/平移/置换不变到 ~1e-7，力经 float64 有限差分验证（6.7e-9）。
